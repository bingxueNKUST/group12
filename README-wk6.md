+ ### 使用案例說明
    | 使用案例名稱 | 雲端廚房點餐 |
    | :---: | :---: |
    | 行動者 | 顧客 |
    | 說明 | 描述點餐過程 |
    | 完成動作 | 1.消費者可根據篩選器來選擇餐點類型或店家  2.點選餐點並前往結帳  3.系統告知結帳成功 |
    | 替代方法 | 1.消費者在搜尋器搜尋餐點或店家  2.點選餐點並前往結帳  3.系統告知結帳成功 |
    | 先決條件 | 消費者至少選擇一項商品並進入結帳 |
    | 後置條件 | 點餐後，可以檢查餐點製作時間與外送抵達時間 |
    | 假設 | 暫無 |
